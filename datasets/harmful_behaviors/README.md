# Harmful behaviors

The directory contains `harmful_behaviors_pair.csv`: behaviors from the PAIR
paper ([Jailbreaking Black Box Large Language Models in Twenty
Queries](https://arxiv.org/abs/2310.08419)), curated from the AdvBench
behaviors ([Universal and Transferable Adversarial Attacks on Aligned Language
Models](https://arxiv.org/abs/2307.15043)).
